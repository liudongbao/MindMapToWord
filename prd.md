# MindMapToWord - 产品需求文档 (PRD)

## 1. 项目概述

### 1.1 项目背景
开发一款 Windows 桌面程序，支持将主流思维导图格式一键转换为排版良好的 Word 文档，完整保留文字、图片、备注、超链接等内容。

### 1.2 目标用户
- 需要将思维导图转换为 Word 文档的个人用户
- 需要批量处理思维导图的企业用户
- 使用 XMind、MindMaster 等工具的思维导图用户

### 1.3 产品定位
一款轻量级、离线运行、功能完善的思维导图转 Word 桌面工具。

## 2. 功能需求

### 2.1 核心功能

| 需求编号 | 功能模块 | 功能描述 | 需求来源 |
| :--- | :--- | :--- | :--- |
| FR-001 | 思维导图解析 | 支持 XMind（.xmind）、MindMaster（.emmx）、FreeMind（.mm/.mmap）、OPML（.opml）、Markdown（.md）格式解析 | 原始需求 |
| FR-002 | Emmx 深度解析 | 针对 MindMaster 特有的 `page/page.xml` 结构，通过 `LevelData/Super V` 建立父子关系，`LevelData/SubLevel V` 按视觉顺序排序 | 补充需求 |
| FR-003 | 内容保留 | 完整保留节点标题、备注/注释、超链接、节点内嵌图片（PNG/JPEG/GIF/BMP/TIFF） | 原始需求 |
| FR-004 | Word 结构化输出 | 按导图层级自动映射到 Word 标题 1~4 + 正文，图片居中且自动等比缩放 | 原始需求 |
| FR-005 | 目录结构保持 | 导出的 Word 文档保持与导图相同的目录结构和节点顺序 | 修复需求 |
| FR-006 | 目录美观度优化 | 最后两层节点（深度 ≥ 3）+ 大段文字（>50 字符）自动渲染为普通正文段落，不进入 Word 目录 | 优化需求 |

### 2.2 批量导出功能

| 需求编号 | 功能模块 | 功能描述 | 需求来源 |
| :--- | :--- | :--- | :--- |
| FR-007 | 多文件选择 | 支持一次选择多个思维导图文件 | 批量需求 |
| FR-008 | 进度条显示 | 转换过程中显示实时进度条和当前处理的文件名 | 批量需求 |
| FR-009 | 错误隔离 | 单个文件转换失败不影响其余文件继续执行 | 批量需求 |
| FR-010 | 汇总结果 | 批量转换完成后显示汇总结果窗口，包含成功/失败数量及详细列表 | 批量需求 |
| FR-011 | 输出目录验证 | 批量导出前验证目标目录可写性，避免权限问题 | 修复需求 |

### 2.3 用户体验功能

| 需求编号 | 功能模块 | 功能描述 | 需求来源 |
| :--- | :--- | :--- | :--- |
| FR-012 | 树状预览 | 解析后在左侧树状视图中显示导图结构 | 原始需求 |
| FR-013 | 统计信息 | 显示节点数、图片数、链接数等统计信息 | 原始需求 |
| FR-014 | 帮助说明 | 内置完整的用户帮助文档，包含快速入门、支持格式、常见问题三个标签页 | 新增需求 |
| FR-015 | 帮助按钮位置 | 帮助按钮位于窗口顶部按钮栏最右侧 | 布局需求 |
| FR-016 | 树结构诊断 | 内置节点层级诊断窗口，显示节点 ID、坐标、排序、原始属性 | 调试需求 |

### 2.4 技术特性

| 需求编号 | 功能模块 | 功能描述 | 需求来源 |
| :--- | :--- | :--- | :--- |
| FR-017 | 离线运行 | Word 生成基于 OpenXML SDK，无需安装 Microsoft Office | 原始需求 |
| FR-018 | 框架版本 | 基于 .NET 8 Windows 桌面应用（WinForms） | 技术要求 |
| FR-019 | 两种发布包 | 提供普通版和单文件版（`_Single.exe`），方便不同场景使用 | 发布需求 |
| FR-020 | 构建脚本 | 提供 `build.bat` 脚本，支持一键编译和发布 | 发布需求 |

## 5. 输出格式规范

### 5.1 Word 文档结构

```
[居中大标题：文件名]

■ 中心主题（标题 1）
  ├ 链接：http://...
  ├ [图片，居中]
  │
  ├ ■ 子主题 A（标题 2）
  │   └ ■ 更细的节点（标题 3）
  │
  └ ■ 子主题 B（标题 2）
      └ 备注：这是一段备注文本
```

### 5.2 目录层级规则

| 导图层级 | Word 样式 | 进入目录 |
| :--- | :--- | :--- |
| 第 1 层 | Heading 1 | 是 |
| 第 2 层 | Heading 2 | 是 |
| 第 3 层及以上（短文本 ≤ 50 字符） | Heading 3/4 | 是 |
| 第 3 层及以上（长文本 > 50 字符，叶子节点） | Normal（正文段落） | 否 |
